# FizzBuzz_TDD_C_Sharp
A small console application using NUnit for unit tests to show the Red > Green > Refactor stages of the well known Kata FizzBuzz.

Uses: 
NUnit v3.6.0
NUnit3TestAdapter v3.6.0
